6.1 計算參數與依據

  • 建築物幾何：高度 $h = {{ chapter_6_data.h|floatformat:2 }} \text{ m}$，屋頂角 $\theta = {{ chapter_6_data.theta|floatformat:1 }}^\circ$。
    • {% if chapter_6_data.path == 'h_le_18' %}
  • 適用規範：$h \le 18 \text{ m}$，依據規範 3.2 節公式 (3.1)。
  • $ p = q(h)[(GC_p) - (GC_{pi})] $
    • {% else %}
  • 適用規範：$h > 18 \text{ m}$，依據規範 3.2 節公式 (3.2)。
  • $ p = q(GC_p) - q_i(GC_{pi}) $
  • 註：對於 $h>18\text{m}$ 之迎風面牆，風壓隨高度 $z$ 變化 ($q=q(z)$)。下表風壓值係以頂層 $z=h$ 處之 $q(h)$ 計算，代表該區域之最大設計風壓包絡值。
    • {% endif %}
  • 計算參數：
    • 平均屋頂高度風速壓 $q(h) = \mathbf{ {{ chapter_6_data.q_h|floatformat:2 }} } \text{ kgf/m}^2$。
    • 內風壓係數 $GC_{pi} = \{ {% for val in chapter_4_data.gcpi %}{{ val|floatformat:3 }}{% if not forloop.last %}, {% endif %}{% endfor %} \}$。
      • {% if chapter_6_data.path == 'h_le_18' %}
    • 最大設計正風壓 $ p^{+} = q(h)[(GC_p)^{+} - (GC_{pi})^{-}] = {{ chapter_6_data.q_h|floatformat:2 }} [(GC_p)^{+} - ({{ chapter_6_data.gcp_i_neg|floatformat:3 }})] $
    • 最大設計負風壓 $ p^{-} = q(h)[(GC_p)^{-} - (GC_{pi})^{+}] = {{ chapter_6_data.q_h|floatformat:2 }} [(GC_p)^{-} - ({{ chapter_6_data.gcp_i_pos|floatformat:3 }})] $
      • {% else %}
    • 最大設計正風壓 $ p^{+} = q(GC_p)^{+} - q_i(GC_{pi})^{-} = q(GC_p)^{+} - {{ chapter_6_data.q_h|floatformat:2 }}({{ chapter_6_data.gcp_i_neg|floatformat:3 }}) $
    • 最大設計負風壓 $ p^{-} = q(GC_p)^{-} - q_i(GC_{pi})^{+} = q(GC_p)^{-} - {{ chapter_6_data.q_h|floatformat:2 }}({{ chapter_6_data.gcp_i_pos|floatformat:3 }}) $
      • {% endif %}
  • 區域寬度 $a$ 值計算：
    • 公式：$a = \max( 0.4h, 0.1 \times \min(B, L) )$，且不小於 $0.9\text{m}$ (或 $0.04\min(B,L)$)。
    • 結果：$a = \mathbf{ {{ chapter_6_data.a_val|floatformat:3 }} \text{ m} }$。
    • (此寬度 $a$ 用於定義牆角區 Zone 5 或屋頂邊緣區 Zone 2, 3 之範圍)
