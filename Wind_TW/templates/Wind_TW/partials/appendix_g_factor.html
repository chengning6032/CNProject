2. 共振反應參數計算

  • 自然頻率 $n_1 = {{ details.fn|floatformat:3 }}\text{ Hz}$
  • 阻尼比 $\beta = {{ details.beta }}$
  • 平均風速 $\bar{V}_{\bar{z}} = \bar{b} (\dfrac{\bar{z}}{10})^\alpha V_{10}(C) = {{ details.V_z_bar|floatformat:2 }}\text{ m/s}$
  • 折減頻率 $N_1 = \dfrac{n_1 L_{\bar{z}}}{\bar{V}_{\bar{z}}} = {{ details.N1|floatformat:3 }}$
  • 共振能量因子 $R_n = \dfrac{7.47 N_1}{(1 + 10.3 N_1)^{5/3}} = {{ details.Rn|floatformat:3 }}$
  • 空氣動力導納函數：
    $\eta_h = 4.6 n_1 h / \bar{V}_{\bar{z}} = {{ details.eta_h|floatformat:3 }} \Rightarrow R_h = {{ details.Rh|floatformat:3 }}$
    $\eta_B = 4.6 n_1 B / \bar{V}_{\bar{z}} = {{ details.eta_B|floatformat:3 }} \Rightarrow R_B = {{ details.RB|floatformat:3 }}$
    $\eta_L = 15.4 n_1 L / \bar{V}_{\bar{z}} = {{ details.eta_L|floatformat:3 }} \Rightarrow R_L = {{ details.RL|floatformat:3 }}$
  • 共振反應因子 $R = \sqrt{\dfrac{1}{\beta} R_n R_h R_B (0.53 + 0.47 R_L)} = {{ details.R|floatformat:3 }}$
  • 共振尖峰因子 $g_R = \sqrt{2 \ln(3600 n_1)} + \dfrac{0.577}{\sqrt{2 \ln(3600 n_1)}} = {{ details.gR|floatformat:3 }}$

3. 陣風反應因子 $G_f$ 計算

依據規範公式 (2.13)，取尖峰因子 $g_Q = g_v = 3.4$：

$$ G_f = 1.927 \left[ \dfrac{1 + 1.7 I_{\bar{z}} \sqrt{g_Q^2 Q^2 + g_R^2 R^2}}{1 + 1.7 g_v I_{\bar{z}}} \right] $$

代入數值：

$$ G_f = 1.927 \left[ \dfrac{1 + 1.7({{ details.I_z|floatformat:3 }}) \sqrt{(3.4)^2 ({{ details.Q|floatformat:3 }})^2 + ({{ details.gR|floatformat:3 }})^2 ({{ details.R|floatformat:3 }})^2}}{1 + 1.7(3.4)({{ details.I_z|floatformat:3 }})} \right] $$

$$ \Rightarrow \mathbf{ G_f = {{ details.final_value|floatformat:3 }} } $$
