4.4.4 中小度地震垂直力係數 (Cz*)
X 方向
A. 計算原始比值:
(不考慮近斷層) $\displaystyle{ \dfrac{S_{aD,V,X}^*}{F_{uD,V,X}^*} = \dfrac{ {{ results.SaDVX_no_fault|floatformat:4 }} }{ {{ results.FuvX_no_fault|floatformat:4 }} } = {{ results.ratio_SaDVX_no_fault_FuDVX_no_fault|floatformat:4 }} }$
B. 依規範修正:
$$ \left( \dfrac{S_{aD,V}}{F_{uv}} \right)_m = \begin{cases} \displaystyle \dfrac{S_{aD,V}}{F_{uv}} & ; \dfrac{S_{aD,V}}{F_{uv}} \le 0.15 \\ \displaystyle 0.52 \dfrac{S_{aD,V}}{F_{uv}} + 0.072 & ; 0.15 < \dfrac{S_{aD,V}}{F_{uv}} < 0.4 \\ \displaystyle 0.70 \dfrac{S_{aD,V}}{F_{uv}} & ; \dfrac{S_{aD,V}}{F_{uv}} \ge 0.4 \end{cases} $$
C. 修正後比值: $\displaystyle{\left( \dfrac{S_{aD,V,X}^*}{F_{uD,V,X}^*} \right)_m = \mathbf{ {{ results.SaDV_over_Fuv_m_X_no_fault|floatformat:4 }} } }$
{% with ratio=results.ratio_SaDVX_no_fault_FuDVX_no_fault %} {% if ratio <= 0.15 %}因比值 {{ ratio|floatformat:4 }} ≤ 0.15，故修正值 = 比值。 {% elif ratio < 0.4 %}因 0.15 < 比值 {{ ratio|floatformat:4 }} < 0.4，故修正值 = 0.52 × {{ ratio|floatformat:4 }} + 0.072 = {{ results.SaDV_over_Fuv_m_X_no_fault|floatformat:4 }}。 {% else %}因比值 {{ ratio|floatformat:4 }} ≥ 0.4，故修正值 = 0.70 × {{ ratio|floatformat:4 }} = {{ results.SaDV_over_Fuv_m_X_no_fault|floatformat:4 }}。 {% endif %} {% endwith %}
