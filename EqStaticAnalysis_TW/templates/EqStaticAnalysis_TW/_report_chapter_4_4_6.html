4.4.6 柱之垂直地震力係數 (Ccolumn)

根據規範解說，柱因與地表一起上下運動，其垂直地震力效應可等效為將其單位長度重量乘以一個額外係數。此係數依地區而異，計算如下：

{% if results.is_near_fault %}
近斷層工址: $\displaystyle{ C_{column} = \dfrac{0.80 S_{DS} I}{3 \alpha_y} = \dfrac{0.80 \times {{ results.SDS|floatformat:4 }} \times {{ results.I|floatformat:2 }} }{3 \times {{ results.alpha_y|floatformat:2 }} } = \mathbf{ {{ results.C_column|floatformat:4 }} } }$
{% else %}
一般區域與臺北盆地: $\displaystyle{ C_{column} = \dfrac{0.40 S_{DS} I}{2 \alpha_y} = \dfrac{0.40 \times {{ results.SDS|floatformat:4 }} \times {{ results.I|floatformat:2 }} }{2 \times {{ results.alpha_y|floatformat:2 }} } = \mathbf{ {{ results.C_column|floatformat:4 }} } }$
{% endif %}
